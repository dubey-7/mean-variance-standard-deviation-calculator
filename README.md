# Mean-Variance-Standard Deviation Calculator

This project calculates the mean, variance, standard deviation, max, min, and sum of a 3x3 matrix using NumPy.

It is part of the FreeCodeCamp Data Analysis with Python certification.

Create a function named `calculate()` that:

- Takes a list of 9 numbers as input
- Converts it into a 3x3 NumPy array
- Returns a dictionary containing:
  - Mean
  - Variance
  - Standard Deviation
  - Maximum
  - Minimum
  - Sum

Each calculation is performed:
- Along rows
- Along columns
- For the entire matrix (flattened)
